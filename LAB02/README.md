วิธีการดำเนินงาน (Methodology)
1. โหลดชุดข้อมูล (Load Dataset)

เริ่มต้นด้วยการนำเข้าไลบรารีที่จำเป็น ได้แก่ Pandas, NumPy, Matplotlib และ Seaborn จากนั้นใช้คำสั่ง pd.read_csv() เพื่ออ่านไฟล์ dataset.csv เข้ามาเก็บในตัวแปร df และใช้ df.head() เพื่อตรวจสอบข้อมูลตัวอย่าง 5 แถวแรกว่าชุดข้อมูลถูกโหลดเข้ามาอย่างถูกต้อง

2. สำรวจข้อมูลเบื้องต้น (Dataset Exploration)

ทำการสำรวจโครงสร้างของชุดข้อมูลเพื่อทำความเข้าใจลักษณะของข้อมูลก่อนนำไปวิเคราะห์ โดยมีขั้นตอนดังนี้

ตรวจสอบขนาดของข้อมูลด้วย df.shape
ตรวจสอบชนิดข้อมูลของแต่ละคอลัมน์ด้วย df.dtypes
แสดงค่าสถิติเบื้องต้น เช่น ค่าเฉลี่ย ค่าต่ำสุด ค่าสูงสุด และส่วนเบี่ยงเบนมาตรฐาน ด้วย df.describe()
ตรวจสอบจำนวนข้อมูลที่หายไป (Missing Values) ด้วย df.isnull().sum()
ตรวจสอบข้อมูลซ้ำ (Duplicate Records) ด้วย df.duplicated().sum()
ตรวจสอบการกระจายของคลาสในคอลัมน์ Survived โดยใช้ value_counts() และแสดงผลเป็นกราฟแท่ง

3. การแสดงผลข้อมูล (Data Visualization)

ทำการสร้างกราฟเพื่อช่วยให้เข้าใจลักษณะของข้อมูลได้ง่ายขึ้น

Histogram

ใช้คำสั่ง df.hist() เพื่อแสดงการกระจายตัวของข้อมูลเชิงตัวเลข เช่น อายุ (Age), ค่าโดยสาร (Fare) และจำนวนพี่น้องหรือคู่สมรส (SibSp)

Correlation Heatmap

ใช้ไลบรารี Seaborn สร้าง Correlation Heatmap เพื่อแสดงความสัมพันธ์ระหว่างตัวแปรเชิงตัวเลข โดยค่าความสัมพันธ์อยู่ระหว่าง -1 ถึง 1 ซึ่งช่วยให้เห็นว่าตัวแปรใดมีความสัมพันธ์กันมากหรือน้อย

4. การทำความสะอาดข้อมูล (Data Cleaning)

ก่อนนำข้อมูลไปใช้งาน จำเป็นต้องปรับปรุงคุณภาพของข้อมูล ดังนี้

Missing Value Handling

จัดการข้อมูลที่หายไป โดย

เติมค่า Age ด้วยค่ามัธยฐาน (Median)
เติมค่า Embarked ด้วยค่าที่พบมากที่สุด (Mode)
เติมค่า Cabin ที่หายไปด้วยข้อความ "Unknown"
Duplicate Removal

ตรวจสอบข้อมูลซ้ำด้วย drop_duplicates() พบว่าชุดข้อมูลไม่มีข้อมูลซ้ำ ดังนั้นจำนวนข้อมูลก่อนและหลังการลบจึงเท่าเดิม

Incorrect Data Correction

ตรวจสอบค่าที่ผิดปกติ เช่น อายุที่มีค่าน้อยกว่า 0 (Age < 0) พบว่าไม่มีข้อมูลผิดปกติในชุดข้อมูล

Data Type Conversion

แปลงชนิดข้อมูลของคอลัมน์ Survived และ Pclass จากชนิดตัวเลขเป็นชนิด Category เพื่อให้เหมาะสมกับการวิเคราะห์ข้อมูล

เปรียบเทียบ Mean และ Median

คำนวณค่าเฉลี่ย (Mean) และค่ามัธยฐาน (Median) ของคอลัมน์ Age เพื่อเปรียบเทียบลักษณะการกระจายของข้อมูล

5. การสร้างคุณลักษณะใหม่ (Feature Engineering)

ทำการเตรียมข้อมูลให้อยู่ในรูปแบบที่เหมาะสมสำหรับการสร้างโมเดล Machine Learning

Label Encoding

ใช้ LabelEncoder แปลงข้อมูลข้อความในคอลัมน์ Sex

Female → 0
Male → 1

เพื่อให้สามารถนำไปใช้กับอัลกอริทึม Machine Learning ได้

One-Hot Encoding

ใช้ pd.get_dummies() แปลงคอลัมน์ Embarked ให้เป็นหลายคอลัมน์ ได้แก่

Embarked_C
Embarked_Q
Embarked_S

ซึ่งช่วยลดปัญหาการตีความลำดับของข้อมูลประเภทหมวดหมู่

สรุปผล

จากการดำเนินงาน ได้ทำการโหลดข้อมูล สำรวจข้อมูล วิเคราะห์ข้อมูลด้วยกราฟ ทำความสะอาดข้อมูล และแปลงข้อมูลให้อยู่ในรูปแบบที่เหมาะสมสำหรับการสร้างโมเดล Machine Learning โดยชุดข้อมูลหลังการประมวลผลไม่มีข้อมูลสูญหาย ข้อมูลอยู่ในรูปแบบที่พร้อมสำหรับการวิเคราะห์หรือการนำไปพัฒนาโมเดลในขั้นตอนต่อไป
